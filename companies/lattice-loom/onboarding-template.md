# Onboarding — New Hire Checklist

> Living doc. Last edited September 2025. — Aaron

Things to do for a new hire:

- [ ] Send offer letter (Nicole)
- [ ] Order laptop (Ops, Apple Business Manager)
- [ ] Set up email + Slack + GitHub (Jason, IT contractor)
- [ ] Add to payroll in Gusto (Nicole)
- [ ] Send welcome email Friday before start
- [ ] Day 1: 9am breakfast with manager, office tour, IT setup
- [ ] Day 1: Lunch with team
- [ ] Week 1: 1:1 with each direct teammate, all-hands intro, comp/benefits walkthrough
- [ ] First 30 days: HIPAA + anti-harassment + SOC 2 training
- [ ] First 60 days: 360 setup if manager
- [ ] 90-day review (Lattice)
- [ ] If on-call: pair on-call week 4-6

— Aaron (last updated before Diane joined; Diane has been doing it differently for Platform hires)
